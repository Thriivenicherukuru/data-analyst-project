# Diabetes Patient EDA Project

## Overview
This project performs Exploratory Data Analysis (EDA) on diabetes health indicator data using Python.

The goal is to identify patterns, correlations, and insights related to diabetes risk factors.

---

## Dataset
Dataset: BRFSS 2015 Diabetes Health Indicators Dataset

Features include:
- BMI
- Smoking
- Physical Activity
- Blood Pressure
- Cholesterol
- General Health
- Age
- Income
- Education

---

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Visualization
5. Insights Extraction

---

## Key Insights
- Higher BMI is strongly associated with diabetes.
- Physical inactivity increases diabetes risk.
- General health condition correlates with diabetes occurrence.
- Age is an important contributing factor.
